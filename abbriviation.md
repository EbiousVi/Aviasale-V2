### Abbreviation ###

**Flight type**
<br/>`dir` = direct, `conn` = connection <br/>
`CFS` = connection flight strategy, `DFS` = direct flight strategy<br/>
`dep` = depart flight, `rtn` = return flight <br/>
**Trip type** <br/>
`ow` = oneway trip, `rt` = round trip <br/>
**Other** <br/>
`pax` = passenger <br/>
`fl` = flight <br/>
`fls` = flights <br/>
`tf` = ticket flight <br/>
`t` = ticket <br/>
